02/07/2006 修改:
==
- 到长安城的客栈拿大顶(dading)可以立即增加 内力和法力最大上限, 免去反复exercise和meditate的烦恼. 并且气血和精神的最大上限
也会被更新, 免去以往必须退出重新进入的麻烦.
- study指令: 加多每次study获得的经验 *random(int*int/100)
- learn指令: 每次learn所耗费的神减半
- practice指令: 加多每次practice获得的经验 *random(int*int/100)
==


02/06/2006 修改战斗系统, 加快玩家或得经验,潜能和道行的速度.
==
- 悟性(int)将更为重要.
- 在战斗中, 获得的经验,潜能和道行可以最多 是原来速度的 (int*int/100) 倍, 因此悟越高, 速度越快
- 现在杀掉一个比自己道行低很多的npc都 有机 会得到少量的道行和潜能, 速度也和你的悟性有关
(下一步将要修改learn,study,practice,exert,meditate等命令加快玩家成长速度,减少练功需要, 速度当然也和悟性有关)
==
